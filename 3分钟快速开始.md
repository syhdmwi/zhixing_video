# 3分钟快速开始

如果你是第一次用这套 skills，不需要先看全部文档。

你只要按下面 4 种情况开口就可以。

## 情况 1：我有文案，想先做图片

直接说：

```text
这是我的文案，先带我走图片流程。
```

系统接下来会带你完成：

1. 选择风格方式
2. 选择图片比例
3. 选择生图模型
4. 确认重复主体
5. 确认主体参考图或三视图
6. 确认正式提示词
7. 批量生图

## 情况 2：我已经有图片，想做视频

直接说：

```text
这些图片已经确认，帮我进入视频生成。
```

系统会继续问你进入哪种视频模式：

- `1` 图转镜头移动视频模式
- `2` 图生视频模式

## 情况 3：我想做数字人

直接说：

```text
我要做数字人，我上传音频和人物视频给你。
```

你不需要自己处理公网链接。
如果你上传的是本地文件，系统会优先自动转存后再提交。

## 情况 4：我想从文案一路做到视频

直接说：

```text
这是文案，帮我从图片一路做到视频。
```

## 你最好准备这些材料

可选但很有帮助：

- 主讲人参考图
- 主讲人三视图
- 主要角色参考图
- 风格参考图

如果没有，也可以先开始。

## 一句最稳的通用说法

如果你完全不知道怎么开始，就直接说：

```text
这是我的文案，帮我从头开始做。
```
