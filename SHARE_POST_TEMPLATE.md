## 标准版

我这边整理了一套 `AI Short Video Skills Suite`，是面向 Codex 的完整 AI 短视频生产技能包，已经打包好，可以直接安装使用。

这套技能包主要覆盖：

- 文案估时和镜头数规划
- 重复主体识别与三视图确认
- 图片提示词生成
- `GPT-Image-2 / nanobanana-2` 双模型生图
- 图生视频
- 图转镜头移动视频
- 文案转音频
- 语音克隆
- 数字人生成
- 最终剪辑整合规划

比较适合下面这几类人：

- 做 AI 口播短视频的人
- 做数字人 + 画面轨双线视频的人
- 想把生图、视频、声音流程标准化的人
- 想沉淀一套可复用工作流的人

这套包的特点：

- 不是单一 prompt，而是多 skill 分工
- 生图时可选 `GPT-Image-2` 或 `nanobanana-2`
- 支持用户提供人物三视图直接锁主角
- 支持主体三视图单独确认
- 图片生成后可进两种视频模式
- 语音克隆和数字人可继续往后衔接

使用前注意：

- 包里不包含真实 API key
- 需要自己配置环境变量
- 数字人测试需要可访问的音频直链和人物视频直链
