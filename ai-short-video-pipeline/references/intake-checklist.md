# Intake Checklist

在用户没有一次性给全信息时，按下面顺序补齐。缺少非关键项时可以使用默认值继续，不必卡住。

## Critical

这几项缺一项就会明显影响产出质量：

- 文案全文
- 目标语言
- 目标画幅
- 主角是谁
- 主角外观是否有参考图

## Strongly Recommended

- 目标平台：抖音、视频号、小红书、YouTube Shorts、B 站等
- 目标时长；若没有则估算
- 视觉风格：写实、电影感、广告感、插画、纪实、轻科幻等
- 生图模型
- 图生视频模型
- 数字人工具
- 剪辑工具
- 是否要拆成“数字人视频轨 + 画面视频轨”

## Default Assumptions

当用户未说明时，默认采用：

- 语言：中文普通话
- 平台：竖屏短视频
- 画幅：`9:16`
- 分辨率目标：`1080x1920`
- 时长估算：中文 `230` 字/分钟，英文 `150` 词/分钟
- 镜头数：默认每分钟 `8～10` 个镜头
- 低密度按 `ceil(duration_seconds / 60 * 8)` 取整
- 普通密度按 `ceil(duration_seconds / 60 * 9)` 取整
- 高密度按 `ceil(duration_seconds / 60 * 10)` 取整
- 默认先按普通密度处理，只有明显满足高密度特征时才升到高密度
- 单镜头时长：`4` 到 `7` 秒
- 画面形态：数字人视频轨和 B-roll 视频轨分开生成，后期再整合
- 剪辑工具：剪映

## User Questions To Ask If Needed

如果需要继续追问，优先问这些，不要一次问太多：

1. 这条视频准备发到哪个平台，画幅是 `9:16` 还是别的比例？
2. 主角有没有参考图，还是只靠文字设定？
3. 你要的视觉风格是什么，偏写实、电影感还是插画感？
4. 你实际会用哪个生图模型、哪个图生视频模型？
5. 数字人是全程主讲，还是只在开头和结尾出镜？
6. 你是否要把数字人视频和画面视频分轨生成后再剪辑整合？

## What To Avoid

- 没有主角设定就开始批量写人物镜头提示词
- 没有画幅就直接生成构图指令
- 没有工具信息却编造平台专有参数
- 没有时长就直接固定镜头总数
- 把数字人轨和图生视频轨混成一个生成步骤
