# User Guidance Templates

这份模板专门用于统一“小白用户引导话术”。

原则：

- 先说用户现在在哪一步
- 再说这一步要做什么
- 最后只给当前必要输入
- 不先解释内部实现

## 1. 总入口模板

当用户还没说清楚自己要做哪一步时，优先用：

```text
你把现有素材发我，我先帮你判断最适合从哪一步开始。

如果你现在是：
1. 有文案，想先做图片
2. 有图片，想继续做视频
3. 想做数字人
4. 想从文案一路做到视频
直接告诉我对应情况就行。
```

## 2. 图片流程开场模板

```text
当前进度：
1. 已收到文案
2. 现在先确定风格、比例和生图模型
3. 下一步会识别重复主体

先选风格方式：
使用风格模版请输入 1
自主设定风格请输入 2
```

## 3. 自主设定风格模板

```text
当前进度：
1. 文案已接收
2. 正在确定风格和画幅
3. 下一步会识别重复主体

你可以直接描述这次想要的图片风格；
如果有风格参考图，也可以上传，并说明“只参考风格”还是“人物也参考”。
图片比例和生图模型也请一起告诉我。
```

## 4. 主体识别确认模板

```text
当前进度：
1. 风格和模型已确定
2. 已完成重复主体识别
3. 下一步会进入主体确认

这是我识别到的重复主体清单：
[在这里列主体]

直接按这份主体清单进入三视图生成请输入 1
需要补充/修改主体请输入 2
```

## 5. 真人参考图两步法模板

```text
主讲人参考图已收到。
这类真人头像为了更像本人，不直接硬扩三视图。

当前进度：
1. 已锁定主讲人真人参考
2. 现在先生成正面动漫化定稿
3. 你确认像本人后，再扩展白底三视图
```

## 6. 主体修改模板

```text
需要修改主体请输入 1，并直接说明“修改哪个主体 + 修改内容”，也可以上传参考图。
不需要修改请输入 2。
```

## 7. 正式提示词确认模板

```text
当前进度：
1. 主体已经确认
2. 正式图片提示词已生成
3. 你确认后就进入批量生图

下面是正式提示词预览：
[在这里展示提示词]
```

## 8. 批量生图完成模板

```text
图片已生成完成，下面是全部结果。

是否满意，有无需要修改的图片？
如果有请输入第几张，并把修改要求一起输入；
如果不需要修改，输入 2 进入视频生成。
```

## 9. 视频阶段入口模板

```text
当前进度：
1. 图片已确认
2. 现在进入视频生成

请选择：
1 图转镜头移动视频模式
2 图生视频模式
```

## 10. 数字人阶段入口模板

```text
当前进度：
1. 现在进入数字人生成
2. 下一步需要音频和人物视频

请上传音频和正面人物视频，我会自动转存并提交数字人生成。
```
