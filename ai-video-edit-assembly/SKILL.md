# AI Video Edit Assembly

## Overview

这个 skill 只负责后期整合。输入是至少两条素材轨：

- 数字人视频轨
- 画面视频轨

## When To Use

- 数字人素材和画面素材已经分别生成
- 用户要做最终成片
- 用户要适配抖音、视频号、小红书

## Core Rules

- 不重新设计前期分镜，除非明显冲突
- 优先保证口播信息传达完整
- B-roll 覆盖不能破坏关键口型同步段
- 平台导向优先考虑竖屏节奏和开头钩子

## Output

至少输出：

- 成片结构概述
- 时间线整合建议
- 每段素材覆盖关系
- 字幕强调点
- BGM/SFX 建议
- 封面与 CTA 建议
- 平台导出建议
