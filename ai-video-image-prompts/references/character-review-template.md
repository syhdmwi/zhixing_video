# Character Review Template

这个模板用于“先给用户看角色方案，再等用户确认”。

## Output Shape

按下面结构输出，不要直接跳去写批量提示词：

```text
【已确认参数】
- 图片数量：
- 图片比例：
- 输出语言：
- 整体风格：

【角色方案】
- 角色定位：
- 性别表现：
- 年龄段：
- 脸型与五官：
- 发型与发色：
- 服装设定：
- 身材与比例：
- 气质关键词：
- 角色标志物：
- 适配该文案的原因：

【统一风格摘要】
- 风格关键词：
- 光线：
- 配色：
- 镜头语言：
- 画面质感：

【请用户确认】
1. 这个角色方向是否满意？
2. 是否需要更年轻 / 更成熟 / 更专业 / 更卡通？
3. 是否要上传参考图来锁定角色？
4. 是否确认进入角色三视图阶段？

【若不满意的修改路径】
- 可修改脸型与五官
- 可修改发型与服装
- 可修改年龄感和气质
- 可改为参考图驱动
```

## Rules

- 这一阶段只做“角色确认”，不写 30 条提示词
- 用户未确认前，不进入三视图
- 用户上传参考图后，应重做角色方案，而不是只微调一句
