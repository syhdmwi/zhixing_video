# GPT-Image-2 Prompt Template 01

这个模板用于 `GPT-Image-2`、OpenAI 图像模型，核心目标是：

- 用更自然的语言描述画面
- 优先保证叙事感和镜头感
- 减少过度模块化、过度表格化的提示词结构
- 仍然保留主体一致性和必要的负面约束

## Core Writing Principle

更适合 `GPT-Image-2` 的提示词，不是先堆大量风格词，而是先把这 3 件事说清楚：

1. 谁是主体
2. 主体正在做什么
3. 主体处在什么环境里

在这三件事清楚以后，再补：

- 风格
- 镜头
- 光线
- 材质
- 一致性约束
- 负面约束

## Recommended Structure

每条正式提示词优先按这个顺序写：

1. 主体
2. 动作 / 表情 / 当前状态
3. 背景与环境
4. 风格
5. 镜头 / 构图
6. 光线 / 时间氛围
7. 材质 / 画面细节
8. 主体一致性约束
9. 负面约束

## Writing Style

推荐写法：

- 用完整自然句
- 更像导演在描述镜头
- 明确告诉模型“谁在什么地方做什么”
- 品牌、logo、地标如果需要明确识别，要直接说清楚

不推荐写法：

- 一上来只堆风格词
- 只有抽象氛围，没有主体和环境
- 像参数表一样一行一个标签、但画面关系不清楚

## Example Skeleton

```text
一个[主体]正在[动作/状态]。
背景是[环境/空间/品牌识别元素]。
[风格]，避免写实风格。
[镜头/构图]。
[光线/时间氛围]。
[材质/细节]。
[主体一致性约束]。
[负面约束]。
```

## Example

```text
一个年轻女生站在未来控制台前，正在讲解 AI 视频赛道的新变化。她神情坚定，黑色长发，穿红色碎花一字肩连衣裙。背景是带有 GitHub 趋势榜和 Hermes agent 项目标识的赛博科技空间。赛博朋克风格，偏动漫化和风格化表达，避免写实风格。中近景，空间透视明显。夜景霓虹边缘光。金属、玻璃、电子屏与全息界面材质。人物样貌和穿着保持与参考图一致。无多余文字，无水印。
```

## Brand And Landmark Rule

如果用户希望观众一眼认出品牌、平台、城市地标或社会常见标识，在 `GPT-Image-2` 提示词里也要直接写清楚，例如：

- 官方 logo 清晰可见
- 品牌识别明确
- 工作区标识清晰可见
- 地标特征明确可识别

不要只写成：

- 有相关元素
- 有品牌氛围
- 有类似某平台的感觉

## Subject Consistency Rule

如果只有一个主主体，可以直接自然地写：

```text
人物样貌、发型、穿着和整体打扮保持与参考图一致。
```

如果有多个主体，不建议在一条正式镜头提示词里同时挂多个主体的三视图强参考。默认应：

- 主主体使用强参考
- 次主体使用文字约束或品牌识别描述

避免把叙事镜头误导成角色设定页或展示页。

## Negative Constraints

默认可直接复用这类表达：

- 无多余文字
- 无水印
- 无杂乱排版
- 避免写实风格
- 避免主体跑形
- 避免人物脸部被过强灯光遮挡
- 避免过暗或过曝

## Best Use Cases

这套模板更适合：

- 人物叙事镜头
- 讲解类镜头
- 情绪镜头
- 品牌对照镜头
- 需要更自然镜头感的正式成图
