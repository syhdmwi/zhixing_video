# Three View Template

这个模板用于“角色确认通过后，先锁定三视图，再做整批提示词”。

## Output Shape

```text
【角色三视图锁定说明】
- 角色名称：
- 角色定位：
- 比例：
- 风格：
- 一致性锁定目标：

【正视图提示词】
...

【侧视图提示词】
...

【背视图提示词】
...

【统一锁定说明】
- 脸部结构保持一致
- 发型保持一致
- 服装保持一致
- 配色保持一致
- 身材比例保持一致
- 标志性元素保持一致

【渲染备注】
- 背景尽量简洁
- 背景默认纯白
- 不做剧情动作
- 以展示角色设定为主
- 便于后续批量剧情图继承
- 不出现任何文字、字母、水印、logo、版式元素
```

## Prompt Rules

三视图提示词应强调：

- 同一个角色
- 同一套服装
- 同一风格
- 干净背景
- 清晰可辨的正/侧/背视角

不要在三视图里加入：

- 大幅动作
- 复杂道具剧情
- 与文案剧情强绑定的场景
- 容易干扰角色锁定的夸张特效
