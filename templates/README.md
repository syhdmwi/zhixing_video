# Templates

这里用于保存 `知行视频skill` 的可复用模板。

模板保存的是跨文案稳定复用的内容：

- 主讲人形象
- 主讲人三视图
- 画面风格
- 风格参考图
- 提示词规则
- 负面约束
- 默认比例
- 默认模型
- 用户明确选择的正式图样例

模板默认不保存：

- 每篇文案临时出现的重复主体
- 用户没有选择的正式图
- 单次项目任务队列

## 命名建议

建议使用清晰的中文模板名，例如：

- `赛博女主持模板`
- `科技海报主讲人模板`
- `商务数字人模板`

文件名可以使用拼音或英文 slug，例如：

```text
cyber-host-template.json
```

## 使用方式

用户下次可以这样唤醒：

```text
使用知行视频skill，套用「赛博女主持模板」帮我做这篇新文案。
```

系统读取模板后，应先展示模板摘要，让用户确认是否沿用。
